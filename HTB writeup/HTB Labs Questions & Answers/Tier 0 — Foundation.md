# Tier 0 — Foundation

### **About Tier 0**

Tier 0 covers the essential building blocks of penetration testing. It introduces users to the basics of connecting to Hack The Box, performing initial enumeration, and interacting with simple services. Each machine in this tier requires one straightforward step to obtain a flag, with guidance and walkthroughs to help users build confidence and become familiar with the process.

### **Skills you’ll gain**

- Connect to a VPN and access remote machines.
- Perform basic port scanning and service enumeration (e.g., using nmap).
- Interact with common network services, such as SSH, HTTP, and FTP.
